# 3D Point Cloud Viewer

Visualizador profissional de nuvens de pontos 3D baseado em Three.js.

## Formatos Suportados
- **LAS / LAZ** (Com suporte a coordenadas GIS de alta precisão e mapa de intensidade)
- **PCD** (Point Cloud Data)
- **PLY** (Polygon File Format)
- **OBJ / FBX** (Modelos 3D)
- **GLB / GLTF** (Formato padrão Web)

## Funcionalidades
- **Otimizador**: Redução de nuvens de pontos gigantes (10GB+) via amostragem inteligente.
- **Visualização**: Controle de tamanho de ponto, cores originais ou customizadas.
- **Precisão**: Tratamento de offsets GIS para evitar "jittering" em coordenadas UTM.
